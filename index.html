Gym Mate - Fitness Companion App

Project Details

Gym Mate is a comprehensive fitness application designed with Figma and built with Flutter. The app helps users track workouts, set fitness goals, follow personalized exercise plans, and monitor progress over time. Key features include:

  • Personalized workout plans based on fitness goals
  • Exercise library with detailed instructions and animations
  • Progress tracking with charts and statistics
  • Workout timer and rest interval management
  • Nutrition tracking and calorie counter
  • Social features to share achievements

Technologies used: Flutter, Dart, Firebase, Provider State Management, Figma (UI/UX Design), Google Fit API integration
